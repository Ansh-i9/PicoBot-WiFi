# PicoBot-WiFi
🤖 Use Raspberry Pi Pico 2W microcontroller 

📝 Important Notes

    Common Ground: You must connect Battery (-) to a Pico GND pin.

    WiFi Band: Use 2.4GHz WiFi. The Pico 2W cannot connect to 5.0GHz.

    Voltage: Approx 2 X 3.7V 2000mAh Battery is perfect for this project.

🚀 Step-by-Step Setup

    Build: Mount Pico, L293D, and TT Motors to your chassis.

    Code: Open Thonny IDE, paste the code, and add your WiFi SSID/Password.

    Install: Save the file to the Raspberry Pi Pico as main.py.

    Run: Power the robot. Find the IP Address in the Thonny terminal.

    Control: Type that IP into your phone's browser and drive!
             you will able to see all the BUTTONS and SPEED SLIDER[0–100% Power (PWM)]
